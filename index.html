File Handling

¿Qué es el File Handling?

El proceso de crear, leer, actualizar y eliminar archivos. Es la forma en que los programas interactúan con datos persistentes.

2 / 13

La Forma Correcta: with open()

La declaración `with` es la forma segura y recomendada. Cierra el archivo automáticamente, incluso si ocurren errores.

with open("mi_archivo.txt", "r") as f:
    # Trabaja con el archivo 'f' aquí
# El archivo ya está cerrado aquí.
3 / 13

Modos de Apertura

El segundo argumento de `open()` es el modo.

'r' - Lectura (Read). Es el modo por defecto. Da error si el archivo no existe.
'w' - Escritura (Write). Crea el archivo si no existe, lo SOBREESCRIBE si existe.
'a' - Añadir (Append). Crea el archivo si no existe, añade contenido al final si existe.
4 / 13

Leer un Archivo Completo

El método .read() lee todo el contenido del archivo en un solo string.

with open("poema.txt", "r") as f:
    contenido = f.read()
    print(contenido)
5 / 13

Leer Línea por Línea

La forma más común y eficiente para archivos grandes es iterar directamente sobre el objeto del archivo.

with open("log.txt", "r") as f:
    for linea in f:
        print(linea.strip()) # .strip() quita saltos de línea
6 / 13

Escribir en un Archivo: 'w'

El modo 'w' borra el contenido anterior del archivo antes de escribir.

with open("saludo.txt", "w") as f:
    f.write("Hola, Mundo.\n")
    f.write("Esta es una nueva línea.")
7 / 13

Añadir a un Archivo: 'a'

El modo 'a' agrega contenido al final del archivo, conservando lo que ya existía.

with open("log_eventos.txt", "a") as f:
    f.write("Nuevo evento registrado.\n")

Rutas de Archivos: pathlib

El módulo `pathlib` es la forma moderna de manejar rutas de archivos de manera robusta.

from pathlib import Path

ruta_archivo = Path("datos") / "reporte.txt"

with open(ruta_archivo, "r") as f:
    # ...

Caso de Uso: Archivos JSON

El módulo `json` es perfecto para guardar y leer estructuras de datos como listas y diccionarios.

10 / 13

Escribir JSON: json.dump()

Guarda un objeto de Python (ej. un diccionario) en un archivo en formato JSON.

import json

datos = {"nombre": "Ana", "edad": 30}

with open("usuario.json", "w") as f:
    json.dump(datos, f, indent=4)
11 / 13

Leer JSON: json.load()

Carga datos desde un archivo JSON y los convierte en un objeto de Python.

import json

with open("usuario.json", "r") as f:
    datos_cargados = json.load(f)

print(datos_cargados["nombre"]) # Ana

Caso de Uso: Archivos CSV

El módulo `csv` facilita la lectura y escritura de archivos de valores separados por comas.

import csv

with open("datos.csv", "r") as f:
    lector_csv = csv.reader(f)
    for fila in lector_csv:
        print(fila) # Cada fila es una lista de strings
